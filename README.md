# 📅 週間予定＆24hタイムテーブル管理アプリ

PCでの一週間の予定把握（一覧性）と、タイムテーブルによる一日の時間管理（タイムボクシング）を融合させた、Flaskベースのタスク・スケジュール管理Webアプリケーションです。

---

## 📝 概要

本アプリは、「今週何をするか」という**大まかなタスク管理**と、「今日それをいつやるか」という**詳細な時間割設計**をシームレスに行き来できるスケジュール管理ツールです。

* **週間ビュー（PC向け）:** 7曜日を横並びで俯瞰し、タスクの追加や曜日移動、メモ書きを1画面で完結。
* **タイムテーブル（モバイル向けUI）:** 1日を24時間のタイムラインに落とし込み、何時に何をすべきかを視覚的にクリアにします。

---

## 🚀 主な機能

### 1. 週間スケジュール管理（メイン画面）
* **クイック予定追加:** タスク名、曜日、優先度（高・中・低）を選択するだけで、該当曜日に即座にタスクを追加できます。
* **インタラクティブ操作（非同期通信）:**
  * **完了トグル:** チェックボックスをクリックすると、画面をリロードせずにタスクが「完了状態（グレーアウト＆打ち消し線）」になります。
  * **簡単曜日移動:** タスクをクリックするとメニューが開き、他の曜日へ一発で移動できます。
  * **テキストファイル風メモ:** タスクごとのメモ欄を展開可能。入力後に枠外をクリック（フォーカスアウト）するか、`Ctrl + Enter` を押すだけで自動保存されます。
  * **ワンクリック削除:** 不要になったタスクは ✕ ボタンで即座に削除できます。

### 2. 24時間タイムテーブル画面
* **未設定タスクのプール:** 週間ビューで追加した「時間が決まっていないタスク」を上部に一覧表示。クリックして時間をセットできます。
* **ビジュアル時間割:** 設定した開始・終了時間（5分刻み）に応じて、24時間のタイムライン上にタスクカードが自動でレンダリングされます。
* **優先度カラーシステム:** 優先度「高（赤）」「中（青）」「低（緑）」に応じてカードの色が変化し、一日の重要度バランスがひと目でわかります。
* **安全なバリデーション:** 開始時間より前の時刻を終了時間に設定した場合、エラーメッセージ（Flashメッセージ）を表示して矛盾を防ぎます。
